# TDD Arrays

This problem set is all about arrays!

## Instructions

Complete each function in `index.js` according to the JSDoc.

Run `npm test` in your terminal to test your code. Try to pass all test cases!

> [!TIP]
>
> In addition to the usual edge cases, make sure you're also thinking about
> array edge cases! What happens if the array is empty, has only one element,
> or has more than one element?
